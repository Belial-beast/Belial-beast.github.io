    # 写在前面 * 首先感谢大佬的项目
    • 网易云音乐在国内是一款很优秀的听歌软件，想必是大家日常用的比较多的一款听歌工具吧，可惜它拥有的版权歌曲真的令人感人，好多歌曲都无法播放，今天这里更新一篇玩机教程：解锁网易云音乐灰色歌曲，希望可以给小伙伴带来帮助~

    脚本原理

    • 利用/ QQ / 虾米 / 百度 / 酷狗 音源替换变灰歌曲链接
    • 为请求增加 X-Real-IP 参数解锁海外限制，支持指定网易云服务器 IP，支持设置上游 HTTP / HTTPS 代理
    • 完整的流量代理功能 (HTTP / HTTPS)，可直接作为系统代理 (同时支持 PAC)

    使用方法

    • 自定义代理 ：填写服务器地址和端口号
    • 代理服务器地址：127.0.0.1
    • 代理服务器端口：2333 （8881）
